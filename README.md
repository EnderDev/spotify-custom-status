## Setup
1. `git clone https://github.com/EnderDev/spotify-custom-status`
2. `cd spotify-custom-status`
3. `npm i`
4. Rename `default_config.json` to `config.json`
5. Enter your discord token in the `config.json`
6. `npm start`
